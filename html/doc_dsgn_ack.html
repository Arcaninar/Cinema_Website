Github Repository: Cinema Website

Design & Technical

  • Structure

    • Here is the sketch for the overall theme layout. In the end, I decided that I would only put 2 links in the Navigation tab (Home, Buy Ticket)
    • There are in total 4 pages for this cinema. In order:
      • Home Page
      • Buy Ticket Page
      • Seat Selection Page
      • Print Page

  • Features

    • All Pages

      • Logo animation
      • Disabled navigation link if it links to the current page

    • Home Page

      • Clicking on the movie thumbnail changes the trailer video to its corresponding movie trailer

    • Buy Ticket Page

      • Selecting a different cinema shows different movie list that is currently playing in the respective cinema
      • Clicking on the movie thumbnail selects it

    • Seat Selection Page

      • A table to show the seat arrangement
      • Hovering over a seat changes its style, whether the seat is selected or not
      • Dynamic sorted seat list text which corresponds to which seat has been selected
      • If the user hasn't selected any seat, clicking the submit button will give a warning text instead of redirecting to the Print Page

    • Print Page

      • Creates a ticket for each seat selected from Seat Selection Page
      • Printing the page will give a different layout

  • Technical Properties

    • Dynamic Listing:

      The list of movies, and cinemas is not statically specified in the HTML files. Instead, the list is put into a javascript file, and for each HTML page which needs the list of movie can use the javascript file. Thus, when adding a movie or cinema or movies in a cinema, can be done by only changing the javascript file, and no further modifying the HTML file is needed

    • Sending information to another page:

      Using both the modified search bar (from using a form and a submit button), and local storage (using a regular button, and javascript "onclick" event listener to redirect the page)

    • Video Outsourcing:

      To display a video on the website, instead of downloading the video, I decided to search for a video online, and link it to that video to save space

    • Refactor:

      Removed redundant codes, and similar codes are replaced with a function or put into a separate javascript file
